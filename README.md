# Knowledge SaaS Platform 🚀

A scalable backend system for a Knowledge-based SaaS platform built using Node.js, Express, TypeScript, and Prisma.

This project implements authentication, role-based authorization, and article management using a clean modular architecture.

---

## 🛠 Tech Stack

- Node.js
- Express.js
- TypeScript
- Prisma ORM
- MySQL
- JWT Authentication
- Role-Based Access Control (RBAC)
